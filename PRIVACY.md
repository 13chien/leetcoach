# Privacy Policy for LeetCoach

Last updated: May 2026

## Overview

LeetCoach is a Chrome extension designed to help users practice coding interviews on LeetCode through contextual hints, guided templates, and lightweight coaching analytics.

## Information Collected

LeetCoach may access and process the following information while the user is on supported LeetCode problem pages:

- Problem title
- Problem description text
- User code editor content
- User interaction analytics such as:
  - hint usage
  - template generation usage
  - run/submit counts
  - session duration
  - code change counts

## How Information Is Used

The collected information is used solely for:

- Generating contextual coding hints
- Generating guided starter templates
- Providing coaching-related analytics
- Improving the coaching experience

Problem and code context may be sent securely to the LeetCoach backend service for AI-assisted hint and template generation.

## Data Storage

LeetCoach stores lightweight analytics locally using Chrome local storage.

LeetCoach does not intentionally collect personally identifiable information.

## Data Sharing

LeetCoach does not sell user data to third parties.

Information is only processed as necessary to provide the coaching features of the extension.

## Third-Party Services

LeetCoach may use third-party AI providers to generate hints and templates.

## Security

Reasonable efforts are made to protect transmitted and stored information. However, no system can guarantee absolute security.

## Changes

This privacy policy may be updated over time as the extension evolves.

## Contact

For questions regarding this privacy policy, please contact the project maintainer through the GitHub repository.
